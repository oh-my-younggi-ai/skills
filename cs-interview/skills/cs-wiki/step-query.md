# Step Query: 위키 질의

위키에 정리된 내용을 찾아 종합해 답한다. 탐색 결과가 가치 있으면 위키에 다시 파일링한다.

## RULES

- 답변은 위키 내용에 근거한다. 위키에 없는 내용을 끌어다 쓸 땐 그 사실을 명시한다.
- `{wiki_root}`는 읽기만 한다 (파일링은 사용자 승인 후).


## INSTRUCTIONS

### 1. 관련 페이지 탐색

1. 루트 `index.md`를 먼저 읽어 관련 주제/페이지를 찾는다 (progressive disclosure).
2. 필요하면 주제 `index.md`로 좁힌 뒤 해당 개념 페이지를 읽는다.

### 2. 종합 답변

찾은 페이지를 종합해 답한다.

- 어느 페이지에서 왔는지 concept ID/링크로 밝힌다.
- 위키에 관련 내용이 **없으면** 솔직히 알리고, "지금 정리할까요?"라고 정리(ingest)를 제안한다.

### 3. 파일링 제안

종합한 결과가 위키에 남길 가치가 있으면(비교표, 새로 발견한 연결 등) 새 페이지로 파일링할지 제안한다 —
탐색이 채팅에 흩어지지 않고 위키에 누적되도록.

- 사용자가 동의하면 `./step-ingest.md`의 절차로 새 페이지를 작성한다.

작업 완료. 추가 요청이 있으면 `./step-01-route.md`로 다시 분류한다.
