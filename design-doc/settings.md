# Design Doc — 스킬 전용 설정

이 파일은 직접 수정해서 이 스킬만의 동작을 조정할 수 있습니다.
공통 설정은 `~/.config/ai-rules/conventions/settings.md`를 따르며, 여기서 같은 키를 정의하면 이 값이 우선합니다.

---

## 출력 디렉토리

```
output_dir: docs/design
```

설계 문서가 저장될 프로젝트 내 디렉토리입니다.

---

## 질문 라운드 제한

```
max_question_rounds: 2
```

정보 수집을 위한 질문 라운드의 최대 횟수입니다.
이 횟수를 넘으면 불명확한 항목이 있어도 사용자에게 진행 여부를 확인합니다.
